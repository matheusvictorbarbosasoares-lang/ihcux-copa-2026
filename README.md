# CopaNaMão

## Integrantes

Matheus Victor Barbosa Soares

## Problema focado

O aplicativo busca resolver a dificuldade dos torcedores em encontrar locais no estádio e evitar filas para alimentação.

## Justificativa de Design

As informações foram organizadas com foco em rapidez e facilidade de leitura, utilizando botões grandes e atalhos rápidos para situações de distração e barulho no estádio.

## Fluxo do Usuário

O usuário acessa a tela inicial, abre o cardápio digital, adiciona itens ao carrinho, realiza o pagamento e recebe o aviso para retirada. Também pode acessar a central de replays para rever lances importantes.
